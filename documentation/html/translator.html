process_pending_instruction() → None[source]#

Process pending Telesoud instruction and convert to robot command.

Converts Telesoud instruction codes to appropriate robot commands: - 0: STOP - 1: GET_ROBOT_DATA - 7: SET_DYNAMIC (speed vector) - 8: START_DYNAMIC - 15: PLAY_CARTESIAN - 16: PLAY_JOINT

Forward robot data to Telesoud interface.
